createNestedComponentTree

open fun createNestedComponentTree(parentContext: ComponentContext, @Nullable component: Component): ComponentTree.Builder

Creates a ComponentTree nested inside the ComponentTree of the provided parentContext. If the parent ComponentTree is subscribed to a LithoLifecycleProvider, the nested ComponentTree will also subscribe to a SimpleNestedTreeLifecycleProvider hooked with the parent's lifecycle provider.

Return

builder for a nested ComponentTree.

Parameters

parentContext

context associated with the parent ComponentTree.

component

root of the new nested ComponentTree.
