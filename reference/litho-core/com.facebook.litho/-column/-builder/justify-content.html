justifyContent

open override fun justifyContent(justifyContent: YogaJustify?): Column.Builder

The JustifyContent property describes how to align children within the main axis of a container. For example, you can use this property to center a child horizontally within a container with FlexDirection = Row or vertically within one with FlexDirection = Column. See https://yogalayout.com/docs/justify-content for more information.

Default: FLEX_START
