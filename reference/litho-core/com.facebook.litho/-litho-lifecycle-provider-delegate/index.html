LithoLifecycleProviderDelegate

class LithoLifecycleProviderDelegate : LithoLifecycleProvider

Default LithoLifecycleProvider implementation. Defines the standard state changes for a Litho ComponentTree. A custom LithoLifecycleProvider implementation can change the lifecycle state but delegate to this to handle the effects of the state change. See an example of how this facilitates a custom lifecycle implementation in AOSPLithoLifecycleProvider.
