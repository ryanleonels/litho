PrepareResult

class PrepareResult

The result of a MountableComponent#prepare or a PrimitiveComponent#prepare call. This will be the Mountable/Primitive this component rendered to, potentially as well as other non-Mountable/Primitive metadata that resulted from that call, such as transitions that should be applied, as well as CommonProps for the component.
