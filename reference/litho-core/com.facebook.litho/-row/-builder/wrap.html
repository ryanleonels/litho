wrap

open override fun wrap(wrap: YogaWrap?): Row.Builder

The FlexWrap property is set on containers and controls what happens when children overflow the size of the container along the main axis. If a container specifies WRAP then its children will wrap to the next line instead of overflowing.

The next line will have the same FlexDirection as the first line and will appear next to the first line along the cross axis - below it if using FlexDirection = Column and to the right if using FlexDirection = Row. See https://yogalayout.com/docs/flex-wrap for more information.

Default: NO_WRAP
