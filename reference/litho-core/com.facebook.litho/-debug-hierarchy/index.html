DebugHierarchy

open class DebugHierarchy

DebugHierarchy provides a light(er) weight way to track and access information about the component parentage of a given MountItem. For a given MountItem, it provides access to a linked list of Class objects representing the class of the Component and each of it's hierarchy parents.
