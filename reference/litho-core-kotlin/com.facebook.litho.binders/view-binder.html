fun Style.viewBinder(binder: RenderUnit.Binder<Any?, View, Any?>): Style

Sets the given binder as a Mount binder for View. The usage of this binder will guarantee that there will be an associated View to it (either because it is wrapping a Mounting View or by creating a wrapping ComponentHost.

Once associated, the binder RenderUnit.Binder#bind can be called at every mount of the render unit associated with this Style, while the RenderUnit.Binder#unbind can be called at every unmount.

The result of the RenderUnit.Binder.shouldUpdate will determine if the bind and unbind methods will be called for any iteration of the mount phase.

For example, if you want your binder to be called at first time the content is mount, and finally the last time it is unmounted before going offscreen, you should set the RenderUnit.Binder.shouldUpdate to false.
